# Article Seeds

`content/seeds/` stores compact publishable ideas discovered during EA development.

This directory is for external-facing material that may later become:

- an X post or thread
- a note article
- a post on the user's site

These files are not full drafts. They are structured seeds built from actual repo evidence.

Use this directory only when the work produced something reusable for readers:

- a strong comparison result
- a failure with a clear lesson
- a decision framework worth sharing
- a strategy or research insight with evidence
- a discipline or judgment lesson that materially affected development

Do not copy raw reports into this directory. Link to the supporting evidence instead.

Suggested filename:

- `YYYY-MM-DD-short-insight.md`
