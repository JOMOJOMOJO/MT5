# Improvement Charter

## Mission

会社自身を改善対象として扱う。

EA の改善だけでなく、次も見直す。

- 部署構成
- shared skills
- shared MCP servers
- 承認フロー
- routing と review cadence

## Default Triggers

- shared skill 追加または削除の直後
- shared MCP 追加または削除の直後
- 組織構造変更の直後
- live 準備へ移る前
- 定期 review 時

## Outputs

- snapshot
- 前回との差分
- review note
- 必要なら CEO approval entry
