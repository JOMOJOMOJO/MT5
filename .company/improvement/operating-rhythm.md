# Operating Rhythm

## Weekly Or Major-Milestone Review

Run a short company review when one of these happens:

- a new EA family is started
- a candidate is promoted from research to serious validation
- a candidate is proposed for demo/live
- a shared skill or MCP changes

Review:

- active queue items
- current skill roster
- current MCP roster
- top reusable findings from the last experiment cycle
- what should be killed, simplified, or promoted next

## Quarterly Review

Run a deeper review even if no org change happened.

Check:

- which skills were actually useful
- which knowledge notes were reused
- which strategy families consumed time without producing a durable edge
- whether org complexity is growing faster than validation quality

## Mandatory Outputs

- refresh `.company/secretary/queue.md`
- add or update an improvement note under `.company/improvement/`
- add one reusable summary under `knowledge/company/` or `knowledge/patterns/`
- record CEO approval if policy-level structure changed
