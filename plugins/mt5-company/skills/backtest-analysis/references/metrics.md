# 指標ガイド

- `Trade count`
  標本数が少ないと、他の指標の説得力が落ちます。
- `Profit factor`
  便利ですが、件数不足や非現実的コスト前提では当てになりません。
- `Expected payoff`
  スプレッドや手数料前提が安定した後の比較に向きます。
- `Max drawdown`
  最終利益だけでなく、絶対値と比率の両方を見ます。
- `Average win / average loss`
  勝率で勝っているのか、損益比で勝っているのかを切り分けます。
- `Long / short split`
  偏りが大きいと、局面依存やフィルター漏れの可能性があります。
- `Time-of-day performance`
  本当の優位性なのか、特定時間帯だけなのかを分けるのに役立ちます。
- `Parameter stability`
  少し値を動かしただけで崩れる戦略は危険です。
- `Out-of-sample behavior`
  最適化ピークを信じる前に必須です。
