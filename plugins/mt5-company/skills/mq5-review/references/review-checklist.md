# レビューチェックリスト

- ロット計算がブローカーの刻みに正規化されているか。
- 注文価格がシンボル桁数に合わせて正規化されているか。
- SL/TP と指値価格が Stops Level と Freeze Level を破っていないか。
- 同じ条件で Tick が連続しても重複エントリーしないか。
- チケット選択が symbol と magic number で適切に絞られているか。
- 失敗時ログに診断できる情報が十分にあるか。
- スプレッド制限や時間帯制限が暗黙ではなく明示されているか。
- テスター専用の近道が本番前提に漏れていないか。
- リスク制御が注文後ではなく注文前に効いているか。
- 決済ロジックが「ポジションなし」と「すでに閉じた後」を安全に扱えるか。
