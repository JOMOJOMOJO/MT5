# Evaluation Ladder

## Default order

1. Short search window
2. Locked validation window
3. Explicit out-of-sample window
4. Demo or forward
5. Live

## Suggested window sizes

- Search: 1 week to 1 month
- Locked validation: 6 to 12 months
- Out-of-sample: 1 to 3 months

## Selection rules

- Optimize only a small set of high-impact inputs.
- Reject passes with low trade count, weak PF, or unstable drawdown.
- Never judge a parameter set only by the search window.
- Record both selected and rejected parameter regions.
